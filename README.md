### Contribute
NyanTTS는 카카오 음성 API 기반의 디스코드 TTS 봇입니다.\
개인 사용 의도로 제작되었습니다.\
디스코드 API Key와 카카오 API Key는 코드에서 제거된 상태로 업로드 되어있습니다.
### Command
=help : 명령어를 확인합니다.\
=connect : 음성 채널에 접속합니다.\
=disconnect : 음성 채널 접속을 해제합니다.\
;[할말] : 여성 목소리로 TTS를 실행합니다.\
,[할말] : 남성 목소리로 TTS를 실행합니다.
